Option A — Generate with AI (iOS 18+)

In the Shortcuts app tap + → New Shortcut, then tap the sparkle / wand icon to open the AI prompt. Copy the prompt below and paste it in.

Create a shortcut that appears in the Share Sheet and accepts any input. Make sure "Use as Quick Action" and "Show in Share Sheet" are enabled in the shortcut settings. Step 1: Get URLs from the shared input. Step 2: Make a GET request to {{ local_ip }}/api/tags?shortcut=true with the HTTP header X-Tether-UUID set to {{ api_uuid }}. Step 3: Repeat with each item in the result. Inside the loop, get the dictionary from each repeat item, then get the value for the key "name" from that dictionary. End the repeat. Step 4: Show a Choose from List using the Repeat Results. Allow selecting multiple items. Step 5: If the Chosen Items contains "+ New", ask for input with the prompt "New categories (comma-separated)". Otherwise set a variable called New Categories to an empty string. End the if. Step 6: Make a POST request to {{ local_ip }}/api/links with headers X-Tether-UUID set to {{ api_uuid }} and Content-Type set to application/json. Set the JSON body with key "url" as a Text field set to the URLs magic variable from step 1, key "tags" as a List field set to the Chosen Item magic variable from step 4, and key "new_tags" as a Text field set to the Provided Input magic variable from step 5. Do NOT use "Ask Each Time" for any of these fields — use magic variables only. Step 7: Show a notification with the title "Tethered!"

The AI may not get every detail right — check the actions it generates against the manual steps below and adjust where needed.

Option B — Manual setup

1
Open the Shortcuts app and create a new shortcut

Tap + in the top-right corner, then tap Add Action.

2
Add: Receive Input from Share Sheet

Search for Receive Input. Leave the input type as Any. This lets the shortcut accept links from any app, including TikTok and YouTube which share as text rather than URLs.

!
Enable "Show in Share Sheet" — do this before building

Tap the shortcut name at the top to open its settings, then tap Add to Share Sheet (or toggle Show in Share Sheet). Without this the shortcut will never appear when you tap Share in other apps. You can do it now or after step 15 — just don't skip it.

3
Add: Get URLs from Input

Search for Get URLs from Input. This extracts a clean URL from whatever was shared — even if the app passed it as text. The input should be Shortcut Input from step 2.

4
Add: Get Contents of URL — fetch your tags

Search for Get Contents of URL. Tap Show More to expand it, then fill in:

URL {{ local_ip }}/api/tags?shortcut=true
Method GET
Headers tap Add new header
Key X-Tether-UUID
Value {{ api_uuid }}
5
Add: Repeat with Each

Search for Repeat with Each. The input should automatically be the Contents of URL result from step 4. The loop runs once per tag object.

6
Inside the loop: add Get Dictionary from Input

Search for Get Dictionary from Input. Set the input to Repeat Item. This converts each raw JSON item into a proper Dictionary that the next step can read.

7
Still inside the loop: add Get Dictionary Value

Search for Get Dictionary Value. Configure it as:

Get Value
for key name
in Dictionary (the result of step 6)
8
Add: End Repeat

Tap End Repeat to close the loop. Shortcuts collects all the names into a Repeat Results variable automatically.

9
Add: Choose from List

Search for Choose from List. Set the input to Repeat Results. Tap the action and enable Select Multiple so you can pick more than one tag.

10
Add: If — check for new category

Search for If. Configure it as:

Input Chosen Item (result of step 9)
Condition contains
Value + New
11
Inside the If: add Ask for Input

Search for Ask for Input. Set it up as:

Prompt New categories (comma-separated)
Input Type Text

The result will be the Provided Input variable — this holds whatever the user types.

12
Add: End If

Tap End If to close the condition. Leave the Otherwise branch empty — if "+ New" wasn't chosen, Provided Input will simply be blank and no new categories will be created.

13
Add: Get Contents of URL — save the link

Add a second Get Contents of URL action. Tap Show More and fill in:

URL {{ local_ip }}/api/links
Method POST
Headers tap Add new header twice
Key X-Tether-UUID → Value {{ api_uuid }}
Key Content-Type → Value application/json
Request Body JSON (already selected)
Tap Add new field → choose Text
Key url → Value: tap the field, choose URLs magic variable (from step 3)
Tap Add new field → choose List
Key tags → Value: tap the field, choose Chosen Item magic variable (from step 9)
Tap Add new field → choose Text
Key new_tags → Value: tap the field, choose Provided Input magic variable (from step 11)

⚠️ For each value field, tap it and pick the magic variable from the list — do not choose "Ask Each Time". If you see a dialog asking for URL, tags, and new_tags when the shortcut runs, open this action and change those fields to magic variables.

14
Add: Show Notification

Search for Show Notification. Set the title to:

Title Tethered!
Body magic variable → URLs (optional)
15
Name and save

Tap the shortcut name at the top and rename it to Tether. Tap Done. It will now appear in your Share Sheet.

How to use it

1
Open any page you want to save

Works in Safari, YouTube, Reddit — anywhere with a Share button.

2
Tap Share → Tether

Scroll down in the Share Sheet until you see Tether.

3
Pick your tags and you're done

Your phone must be on the same Wi-Fi network as the server.
